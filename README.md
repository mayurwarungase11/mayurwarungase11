# 💫 About Me:
<br>     My name is Mayur Warungase, and I am from Nashik. I have completed my Bachelor’s degree in Science from Pune University and have 9 months of experience as a DevOps Intern. I have a strong foundation in Linux, including user management, file permissions, process management, cron jobs, and file editing with Vim. I have hands-on experience with AWS core services such as EC2, VPC, IAM, S3, RDS, Route 53, Lambda, CloudWatch, EFS, and CloudFront. I have also worked with tools like Git and GitHub for version control, Docker for containerization, Kubernetes for orchestration, Terraform for infrastructure automation, Jenkins for CI/CD pipelines, and Datadog for monitoring and observability. I am passionate about building efficient, automated, and scalable systems, and I enjoy learning new technologies and applying best practices to improve system reliability.
